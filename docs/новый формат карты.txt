модификация карты.

карта начинается с 20 адреса.
на каждое устройство выделяется по 4 регистра (по 8 бит)
первые 2 регистра - это "чистые данные" с устройства
вторые 2 регистра - это данные для устройства с морды


устройства в сети модбас адресуются с 1 и максимальный адрес в пределах системы может быть 30

первые 4 регистра общего назначения и управляются только веб-мордой. самый первый из этих регистров - это время (первый байт - минуты, второй байт - секунды)


т.е. 	20 регистр - время
	21 регистр - служебный
	22 регистр - служебный
	23 регистр - служебный

	24 регистр - данные с устройства с id 1
	25 регистр - данные с устройства c id 1
	26 регистр - данные ДЛЯ устройства с id 1
	27 регистр - данные ДЛЯ устройства с id 1

	28 регистр - данные с устройства с id 2
	29 регистр - данные с устройства c id 2
	30 регистр - данные ДЛЯ устройства с id 2
	31 регистр - данные ДЛЯ устройства с id 2

как теперь работает мастер:
	считывает время из файла времени от морды и пишет в карту
	для каждого устройства в файл карты вписывает даные с устройства и считывает из файла данные для устройств в карту


	т.е. из приведенного выше примера в файл карты будут вписаны регистры 24 25 28 29 и считаны все остальные.
